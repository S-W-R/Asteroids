Описание
Данное приложение является вариацией на тему «Asteroids» 


Требования
Python 3.7.4
PyQt версии 5

Запуск:
main.py
Пример запуска: ./main.py

Управление:
W - Включить двигатель
A - Поворот направо
D - Поворот налево

Space - Использовать способность 1
RControl - Использовать способность 2
